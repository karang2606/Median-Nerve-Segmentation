# Median-Nerve-Segmentation

## Installation

Clone the git-repository first.
```
git clone https://github.com/karang2606/Median-Nerve-Segmentation.git
```
Set directory structure as follows:
```
VisTR
├── data
│   ├── train
│   |   ├──patient_01
│   |   |   ├──images
│   |   |   |   ├──0.jpg, ....
│   |   |   ├──masks
│   |   |       ├──0.png, ....
│   ├── test
...
```
